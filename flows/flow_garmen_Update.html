📌 CATATAN PENTING — Berlaku di Seluruh Fase Produksi:
• Setiap LHK WAJIB terkoneksi dengan nomor SPK di sistem
• LHK Proses Sampel WAJIB terkoneksi dengan MAP (bukan hanya SPK)
• SPK WAJIB terkoneksi dengan SO di sistem
• BS Recording per proses (Cutting, Cetak, Bordir, Sewing, Lipat, Koli) harus diinput di setiap LHK
• QC di setiap proses (IPQC) adalah rekomendasi audit untuk standar kualitas lebih baik
• SPK Close ≠ SO Close: SPK Close setelah SJ terbit, SO Close setelah bayar lunas
